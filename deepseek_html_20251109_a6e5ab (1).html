Part 1: What is a Linear Equation in One Unknown?

A linear equation in one unknown is an equation with only one variable (like x) that we need to solve. When we solve it, we find the value of x that makes the equation true.

These equations look like:

x + a = b and ax + b = cx + d etc.

as long as there exists one unknown and the index of the unknown is 1.

Examples of Linear Equations:

  • 2x - 4 = 5 + x
  • 3x + 2 = 8 - x
  • 5x = 15
  • x + 7 = 2x - 3
  • 4 - 2x = 3x + 9

Solution on a Number Line

2x - 4 = 5 + x
Solution: x = 9

Part 2: Linear vs Non-Linear Equations

Compare how linear equations have one solution while non-linear equations can have curves

Linear Equation

One solution: x = 9

Non-Linear Equation

Part 3: Checking if a Point is a Solution

See if a number makes the equation true

To check if a number is a solution to an equation:

  1. Replace the variable with the number
  2. Calculate both sides of the equation
  3. If both sides are equal, the number is a solution!

Example:

For the equation 2x - 4 = 5 + x:

  • If x = 9: 2(9) - 4 = 18 - 4 = 14 and 5 + 9 = 14 ✓
  • If x = 5: 2(5) - 4 = 10 - 4 = 6 and 5 + 5 = 10 ✗
